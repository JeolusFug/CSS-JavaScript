변형

변형(transform, 트랜스폼)은 특정 요소의 크기나 형태 등 스타일이 바뀌는 것


x, y를 주고 2차원 변형이나 z축(앞뒤)를 주어 3차원 변형이 가능하다


시각적인 효과를 주기 위함


위치를 옮길때는 translate


크기를 조절할때는 scale


커서를 위로 옮겨서 확인 가능


회전시킬때에는 rotate 양수 음수에 따라 시계 반시계 방향으로 회전



입체감 있는 움직임도 가능하다



perspective: 200px; 을 추가해 주어야 입체감 있는 깊이감 있는 움직임이 가능하다 0보다 큰 값을 사용해야함


부모 요소에 지정을 해주어야함


가장 밑, 자식요소인 rotatex 쪽에 입체감을 줄 때에는


그 위의 부모요소인 origin2에 perspective를 주어야 한다는 뜻


transition: all 3s;은 몇초동안 움직임을 완성시킬지


3s는 3초동안 움직인다는 의미


비트는 것도 가능



트랜지션 효과

요소의 색상 모양, 혹은 요소의 테두리의 색상, 모양등을 애니매이션 효과처럼 서서히 변하게 하는 것


transition-property: width, height; 어떤 속성에 transition 효과를 줄 것인지


width: 200px; height: 120px; 어떻게 효과를 줄것인지


transition-duration: 2s, 1s; 두개의 속성을 몇초동안 변화시킬건지


transition-duration: 2s 이렇게 사용하면 두개 동일하게 2초동안 변화한다


transition-delay: 1s;는 변화하기 전에 1초동안 딜레이를 준다


앞의 코드나 다른 행동이 있을때 기다린 후에 사용할 수 있도록 해준다


transition-timing-function: linear; 이건 변화의 시작 중간 끝부분의 속도를 지정해준다


아래 비교 가능


ease
linear
ease-in
ease-out
ease-in-out

transition: 2s ease-in; 이라고 줄여서 사용도 가능하다


사용가능한 속성값은 all(어떤거 할지)(all은 생략 가능) 몇초동안 움직일지 딜레이는 몇초 줄건지 어떤 움직임을 할건지 4개를 설정 가능하다


transition: all 2s 1s ease-in; 이면 모든 요소를 1초뒤 2초동안 ease-in으로 변화시킨다는 의미


아래처럼 다 넣을수도 있다


애니매이션 효과

특정지점에서 스타일을 바꾸며 애니메이션 효과를 준다


이 특정지점을 키 프레임이라고 한다


키프레임마다 이름을 주어서 사용한다


요소에 animation-name을 정의해주고


@keyframes 이름 을 넣어 사용한다


시작 부분을 from 끝나는 부분을 to로 해서 변형 가능하다 transparent는 투명하다는 의미


animation-duration은 애니메이션을 몇초동안 할거다


animation-iteration-count: infinite; 는 애니메이션을 몇번 반복할 것인지 지금은 무한반복


이 속성들을 한꺼번에 사용도 가능


animation: rotate 1.5s infinite, background 1.5s infinite alternate; 이라고 표현도 가능한대


rotate 라는 애니메이션을 1.5초동안 무한히, background라는 애니메이션을 1.5초동안 무한히


위와 같이 여러개의 애니메이션 효과를 같이 줄수있다


rotate가 끝난 뒤 background가 일어나는게 아니라


두개가 동시에 같이 일어나는 것

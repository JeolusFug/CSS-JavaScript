글자와 관련된 속성

font-family는 글꼴 종류를 지정


내 컴퓨터에는 있는대 보는 사람의 컴퓨터에는 없다면 제대로 표시되지 않을 수 있음


특정 폰트가 필요하다면 폰트를 다운로드 할 수 있도록 제공해주면 됨


font-size는 글자 크기를 지정


font-style은 글자를 이텔릭체로 표시할지 지정


font-weight는 글자의 굵기를 지정


웹폰트는 특정 폰트를 사용할 때 폰트를 웹서버에 올려놓고 사용자가 웹 사이트에 접속했을때 폰트도 다운로드해서 사용하도록 하는것


네트워크의 영향을 받고, 웹폰트가 많으면 많을수록 느려질 가능성이 있음


font-family에 글꼴을 두개 이상 설정하는것은 첫번째를 받지 못했을 때


두번째 글꼴을 사용한다는 뜻임


색상

색상 이름 표기법은 잘 알려진 색상 이름으로 표시함


16진수표기법은 #을 쓰고 뒤에 16진수 6자리를씀 #ffffff, #000000과 같이 사용


두자리씩 빨강, 초록, 파랑의 양 두자리가 겹치면 줄여쓰기 가능


#00ffcc -> #0fc


hsl/hsla 표기법은 세가지 숫자로 표시하고 색상, 채도, 밝기의 양


투명도를 조절할때는 마지막에 알파값을 추가


rgb/rgba 표기법은 세가지 숫자로 표시하고 빨강, 초록 파랑의 양 0~255


투명도를 조절할 때는 마지막에 알파값을 추가함


마우스 커서를 올리면 팔레트가 나옴


이름부분을 누르면 종류가 바뀌니까 선택할 수 있음


텍스트 스타일 속성

color는 글자색 지정


text-decoration 은 텍스트에 밑줄 취소선을 설정 하이퍼링크를 만들면 밑줄이 생기는대 이걸 없앨때 많이 사용 none을 사용하면 없어짐


text-transform 텍스트 전체 혹은 첫글자를 대문자로 설정


text-shadow 텍스트에 그림자를 추가. 가로, 세로, 번짐정도, 색상 설정 가능


letter-spacing 글자 사이의 간격을 지정


word-spacing 단어 사이의 간격을 지정


text-align 텍스트 정렬 방법을 지정


line-height 줄 간격을 조절 px보다 그냥 숫자를 넣어 몇배의 단위로 사용하는게 좋음

  2. 간단하게 list-style이라고도 사용 가능
  3. 이 기호를 없애버릴수도 있음

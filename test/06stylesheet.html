CSS, 스타일시트, 스타일

코드와 스타일을 분리하는 것은 중요하다


코드에서 틀린 부분을 찾기 위해 스타일과 코드를 모두 보지 않아도 되기 때문이다


또 현재 여러기기들이 인터넷에 접근하는데


기능과 크기 모두 다르기 때문에


반응형 웹 디자인을 위해 분리한다


그냥 웹에서만 봐도 크기를 바꾸면 디자인과 화면 배치가 달라진다


이를 위해서 스타일 시트를 이용한다


스타일은 각각 하나하나를 의미하고


스타일시트는 문서안에 사용된 스타일을 모아놓은걸 스타일시트라고 한다


스타일시트는 내부 외부로 나뉘는대 문서안에 만들어 버릴수도 있고


외부에서 파일로 만들고 링크로 연결할 수도 있다.


내부는 헤드에 스타일을 정의한다


인라인 스타일도 있는대, 테그에 직접 적는 것이다.


하지만 코드가 길어질수록 고치기 어려워지때문에 절대 바꿀일이 없다거나 너무 급할때만 사용함


h2 {
color: red;
font-size: 3.5em;
font-style: italic;
}


에서 h2를 선택자, color를 속성, red를 속성 값이라고 한다


외부 스타일은 stylesheet2에서 계속
